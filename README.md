## 📥 导入步骤

### 方法一：Chrome / Edge / Brave

1.**打开书签管理器**

- 快捷键：`Ctrl + Shift + O` (Windows/Linux) 或 `Cmd + Shift + O` (Mac)
- 或点击浏览器右上角 `⋮` → `书签` → `书签管理器`

2.**导入书签**

- 点击右上角 `⋮` 菜单
- 选择 `导入书签`
- 在弹出的文件选择器中，选择 `charity_site_bookmark .html`
- 点击 `打开`

3.**验证导入**

- 在书签栏中应该能看到 `收藏夹栏` 文件夹
- 展开后可以看到 `公益站` 及其子分类

---

### 方法二：Firefox

1.**打开书签库**

- 快捷键：`Ctrl + Shift + B` (Windows/Linux) 或 `Cmd + Shift + B` (Mac)
- 或点击菜单栏 `书签` → `管理书签`

2.**导入书签**

- 点击顶部菜单 `导入和备份`
- 选择 `从 HTML 文件导入书签...`
- 选择 `charity_site_bookmark .html` 文件
- 点击 `打开`

3.**整理书签**

- 导入的书签会出现在 `书签菜单` 或 `其他书签` 中
- 可以拖拽到 `书签工具栏` 方便访问

---

### 方法三：Safari (macOS)

1.**打开书签编辑器**

- 点击菜单栏 `文件` → `导入自` → `书签 HTML 文件...`

2.**选择文件**

- 在文件选择器中找到 `charity_site_bookmark .html`
- 点击 `导入`

3.**查看书签**

- 按 `Cmd + Shift + B` 打开书签侧边栏
- 导入的书签会出现在 `导入的书签` 文件夹中

## ⚠️ 注意事项

### 1. 书签合并问题

- 导入操作会**追加**书签，不会覆盖现有书签
- 如果重复导入，会产生重复的书签条目
- 建议导入前先备份现有书签

### 2. 文件名问题

- 原文件名包含空格：`charity_site_bookmark .html`
- 某些系统可能识别为 `charity_site_bookmark` 和 `.html` 两个部分
- 如果导入失败，尝试重命名为 `charity_site_bookmark.html`（去掉空格）

### 3. 站点可用性

- 公益站可能随时关闭或更换域名
- 建议定期检查链接有效性
- 失效站点及时清理，避免书签混乱

### 4. 安全提醒

- 部分站点需要注册账号，注意保护个人信息
- 不要在公益站使用与重要账号相同的密码
- 谨慎使用需要绑定支付方式的站点

## 🆘 常见问题

### Q1: 导入后找不到书签？

**A:** 检查以下位置：

- Chrome/Edge：`书签栏` → `收藏夹栏`
- Firefox：`书签菜单` 或 `其他书签`
- Safari：`导入的书签` 文件夹

### Q2: 导入失败提示格式错误？

**A:** 可能原因：

1. 文件编码问题（应为 UTF-8）
2. 文件损坏，尝试用文本编辑器打开检查
3. 浏览器版本过旧，更新到最新版本

### Q3: 书签重复了怎么办？

**A:** 手动删除重复项：

- Chrome/Edge：书签管理器中按住 `Ctrl` 多选删除
- Firefox：书签库中可以按名称排序，方便找到重复项

### Q4: 如何批量检查链接有效性？

**A:** 推荐工具：

- Chrome 扩展：`Check My Links`
- Firefox 扩展：`LinkChecker`
- 在线工具：`Dead Link Checker`
